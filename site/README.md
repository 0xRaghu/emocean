# Emocean Website

Landing page for [emocean.dev](https://emocean.dev) — the live campfire where embers glow.

## Features

- **Live campfire grid** — Real embers fetched from the API
- **Card rotation** — Random ember swaps every 5 seconds with fade animations
- **Toss form** — Submit anonymous messages with tag selection
- **Real-time stats** — Total embers, stokes, and active developers

## Development

```bash
pnpm install
pnpm dev          # localhost:4321
pnpm build        # Production build
```

## Deployment

Deployed on Vercel at [emocean.dev](https://emocean.dev).

## Tech

- Astro
- Tailwind CSS
- DiceBear avatars
